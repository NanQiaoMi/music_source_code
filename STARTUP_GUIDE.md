# Vibe Music Player 项目启动指南

欢迎使用 **Vibe Music Player**，这是一个基于 Next.js + Electron 构建的专业级发烧友音乐播放器。

## 🛠️ 环境准备

在启动项目之前，请确保您的系统中已安装以下环境：

1.  **Node.js**: 建议版本 v18.x 或更高
2.  **Python**: 建议版本 3.8 或更高（用于后端 API 服务）
3.  **Git**: 用于版本管理

## 📦 安装依赖

在项目根目录下运行以下命令安装 Node.js 依赖：

```bash
npm install
```

如果需要使用后端功能（如歌词解析、API 增强），请进入 `backend` 目录安装 Python 依赖（如果存在 `requirements.txt`）：

```bash
cd backend
# 建议使用虚拟环境
python -m venv .venv
source .venv/bin/activate  # Windows 使用 .venv\Scripts\activate
pip install -r requirements.txt
cd ..
```

## 🚀 启动项目

本项目包含前端、后端和桌面端（Electron）三个部分。

### 1. 快速启动（推荐）

如果您是在 Windows 环境下，可以直接运行根目录下的脚本：

- **`start-full.bat`**: 一键启动后端 Python 服务和前端 Next.js 服务。

### 2. 分步启动

#### 启动后端 (Python)
```bash
cd backend
python -m uvicorn main:app --reload --port 8000
```
后端 API 将运行在: `http://localhost:8000`
API 文档地址: `http://localhost:8000/docs`

#### 启动前端 (Next.js)
```bash
npm run dev
```
前端应用将运行在: `http://localhost:3025`

#### 启动桌面端 (Electron)
```bash
npm run dev:electron
```
这将同时启动 Next.js 服务并打开 Electron 窗口。

## 🏗️ 项目打包

### 打包前端静态文件
```bash
npm run build
```

### 打包 Windows 桌面版程序 (.exe)
```bash
npm run build:electron:win
```
打包后的文件将生成在 `dist-electron/` 目录下。

## 📂 项目结构说明

- `src/`: 前端源代码 (React/Next.js)
  - `components/`: UI 组件
  - `hooks/`: 自定义 React Hooks
  - `store/`: 状态管理 (Zustand)
- `electron/`: Electron 主进程与预加载脚本
- `backend/`: Python 后端服务
- `public/`: 静态资源文件
- `scripts/`: 开发辅助脚本

## ❓ 常见问题

- **端口冲突**: 如果 3025 或 8000 端口被占用，请运行 `npm run clean:port` 释放端口。
- **渲染问题**: 本项目使用了大量的 Canvas 和 Web Audio API，建议在支持硬件加速的环境下运行。
