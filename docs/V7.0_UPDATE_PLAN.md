## 🎯 详细更新计划

### 阶段 1：音频特效实际处理（高优先级）

#### 任务 1.1：完善 AudioEffectsManager
**文件**: `src/lib/audio/AudioEffectsManager.ts`

**目标**: 让音频特效真正对播放的音乐产生效果

**具体步骤**:
1. 修改 `connect()` 方法，正确连接到现有音频图
2. 在 `useAudioPlayer` 中集成 AudioEffectsManager
3. 实现特效参数变化时的实时更新
4. 实现 5 种特效的完整处理：
   - ✅ 混响（Reverb）- 卷积混响
   - ✅ 立体声增强（Stereo）- 立体声展宽
   - ✅ 响度平衡（Loudness）- 动态压缩
   - ✅ 人声消除（Vocal Remove）- 中置声道消除
   - ✅ 低音增强（Bass Boost）- 低频 shelving 滤波

**验收标准**:
- [ ] 所有 5 种特效都能真正改变音频输出
- [ ] 特效参数调节实时生效
- [ ] 特效开关立即生效
- [ ] 无音频卡顿或爆音
